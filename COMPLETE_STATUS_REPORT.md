## HOW IT WORKS (IMPLEMENTATION DETAILS)

### Fetching

**Web pages** (iOS, Android):
- Use native `fetch` with timeout (10 seconds)
- Parse HTML with `cheerio` package
- Rate limited (10 requests/min) with `bottleneck`
- Cached (90 days) to avoid repeated fetches

**APIs** (GitHub, Stack Overflow):
- GitHub Search API: Search issues with feature keywords
- Stack Exchange API: Search questions with feature keywords
- Rate limited (respect API limits)
- Optional tokens for higher limits

**Guidelines documents** (App Store, Play Store):
- Fetch complete document once
- Cache for 30 days
- LLM searches document for relevant sections

---

### Parsing

**HTML to text**:
```typescript
const $ = cheerio.load(html);
const text = $('.content-section').text();
```

**API responses to edge cases**:
```typescript

  scenario: issue.title,
  description: issue.body
}));
```

**LLM extraction**:
